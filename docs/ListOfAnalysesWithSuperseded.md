# List Of Analyses 3.0.0 (including superseded and fastlim results)
List of analyses and topologies in the SMS results database, comprising 7736 individual maps from 1529 distinct signal regions, 130 different SMS topologies, from a total of 159 analyses.
The list has been created from the database version `3.0.0.`
Results from FastLim are included. There is also an  [sms dictionary](SmsDictionary) and a [validation page](Validation).
Link to list of results [without superseded and fastlim results](ListOfAnalyses).

A standard normal distribution is expected if no new physics is in the data. New physics would manifest itself as an overabundance of large (positive) significances.</sub></p>
