# Validation plots: comparison of best-SR vs SR combination
This page collects all validation plots that show comparisons of statistical models:
best signal region only versus statistical models at varying degrees of complexity.
The list has been created from the database version 3.1.2fast, considering also superseded results.
Go to [ATLAS 13 TeV results](#ATLAS), [CMS 13 TeV results](#CMS).
There are no 8 TeV results with statistical models.

There is also a [ListOfAnalyses312fast](https://smodels.github.io/docs/ListOfAnalyses312fast), 
a [ListOfAnalyses312fastWithSuperseded](https://smodels.github.io/docs/ListOfAnalyses312fastWithSuperseded), and [Validation312fast](Validation312fast).
